== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-product-table-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add your block to any page
